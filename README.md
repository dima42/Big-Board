## Config variables

Define these locally and Heroku.

- `BIG_BOARD_DB_HOST`
- `BIG_BOARD_DB_NAME`
- `BIG_BOARD_DB_USERNAME`
- `BIG_BOARD_DB_PASSWORD`
- `GOOGLE_CLIENT_SECRET`
- `TOBYBOT_SLACK_KEY`
- `BIGBOARDBOT_SLACK_KEY`
- `PALINDROME_SLACKBOT_TOKEN`
